# SAPZ Platform Source Code Repository

Welcome to the source code repository for the **Special Agro-Industrial Processing Zones (SAPZ)** Enterprise Platform.

---

## 📁 Repository Structure

```
sapz-platform/
├── backend/          # Laravel 11 / PHP 8.3 Enterprise API & Application Services
├── frontend/         # React / Next.js / TailwindCSS Enterprise Management Portal (ESM)
├── mobile/           # Flutter / React Native Field Data Collection & Mobile Client
├── infrastructure/   # Docker Compose, Kubernetes Manifests, Ansible & Terraform Scripts
├── database/         # PostgreSQL / MySQL Migrations, Seeders & Data Dictionary
├── docs/             # Technical Documentation, OpenAPI Specs & Developer Guides
├── work-packages/    # Implementation Work Packages (WP-01 through WP-N)
├── prompts/          # AI Engineering Execution & Review Prompts
├── architecture/     # System Architecture Diagrams, Schemas & ADRs
├── tests/            # Automated Unit, Integration, E2E & Security Test Suites
├── monitoring/       # Prometheus, Grafana, ELK Stack & Health Check Configurations
├── scripts/          # Build, Deployment, Seed & Maintenance Utility Scripts
└── .github/          # CI/CD Workflows, Pull Request Templates & Issue Templates
```
